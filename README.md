# Space Fleet API

REST API для управления космическим флотом: кораблями, миссиями и экипажем.

## Возможности

* Управление космическими кораблями (CRUD)
* Управление миссиями
* Добавление кораблей в миссии
* Обработка ошибок (400, 404)
* Чистая модульная архитектура (как в продакшене)

---

## Архитектура

Проект разделён на слои:

* **models/** — бизнес-логика и сущности
* **routes/** — HTTP эндпоинты (Flask Blueprint)
* **storage.py** — временное хранилище (вместо БД)
* **errors.py** — обработка ошибок
* **app/** — инициализация приложения

---

## Установка

```bash
git clone <repo>
cd space_fleet
pip install -r requirements.txt
```

---

## Запуск

```bash
python run.py
```

Сервер будет доступен по адресу:

```
http://127.0.0.1:5000/api/v1
```

---

## 📡 API Эндпоинты

### Spaceships

| Метод  | URL              | Описание             |
| ------ | ---------------- | -------------------- |
| GET    | /spaceships      | Получить все корабли |
| POST   | /spaceships      | Создать корабль      |
| PUT    | /spaceships/{id} | Обновить статус      |
| DELETE | /spaceships/{id} | Удалить              |

---

### Missions

| Метод | URL                               | Описание         |
| ----- | --------------------------------- | ---------------- |
| GET   | /missions                         | Получить миссии  |
| POST  | /missions                         | Создать миссию   |
| POST  | /missions/{id}/add_ship/{ship_id} | Добавить корабль |

---

## Пример запроса

```bash
curl -X POST http://127.0.0.1:5000/api/v1/spaceships \
-H "Content-Type: application/json" \
-d '{"name": "Explorer", "type": "research"}'
```
